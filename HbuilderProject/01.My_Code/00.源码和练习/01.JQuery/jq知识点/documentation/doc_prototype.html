jQuery包装集

● CSS选择器 为了使样式(css)和结构(html)分离，就要在外部样式表中能够引用到页面元素：CSS选择器
● jQuery包装集 jQuery的其中一个目的，就是方便的访问&操作DOM元素，jQuery也需要一种方法，把jQuery代码和页面元素连接：jQuery包装集 jQuery通过$(selector)的方式，可以轻松的获取到任何DOM元素。其中，selector为字符串，字符串内容为一个css选择器
举个栗子：$('p a')将返回页面中所有p 下面的a标签
● jQuery包装集方法调用 $() 函数，返回一个特别的JavaScript对象，它包含着与选择器相匹配的DOM元素的数组(包装集)。这个对象(包装集)拥有大量的方法，可以方便的操作元素。 
//举个栗子：
//将所有p下面的a标签，追加一个red的class
$('p a').addClass('red');
● 包装集和元素之间的转换 包装集 => DOM元素：从包装集数组中取出的元素，即为DOM元素 。例如：$('#submit')[0] DOM元素 => 包装集：通过$, 可以把DOM元素转为包装集。例如: 
var btn = document.getElementById('submit');
$(btn).addClass('focus');
● 链式操作： 在jQuery的包装集上执行一个函数之后，可以在这个包装集上，连续的执行其他操作。
例如： 
$('.red')
    .removeClass('red')
    .addClass('green')
    .css('border', 'pink solid 2px');
● 包装集常用方法 addClass / removeClass / hasClass 
$('.red').removeClass('red');
 click 
$('.#btn').click(function() {
    alert('hello jquery');
});
css 
$('.red').css(''background, 'red');
 html 
$('.red').html()
 show / hide 
$('.red').show()
