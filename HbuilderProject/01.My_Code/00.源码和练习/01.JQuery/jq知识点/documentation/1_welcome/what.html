什么是jQuery

● jQuery is a fast, small, and feature-rich JavaScript library.
It makes things like HTML document traversal and manipulation,
event handling, animation, and Ajax much simpler with an easy-to-use
API that works across a multitude of browsers. (http://api.jquery.com/)

● jQuery是一个，快速、轻量级、功能强大的JS库。它使得访问和操作DOM、
事件处理、动画、Ajax等操作更易于使用，且可隐藏跨平台的细节

● jQuery的组成：
--- DOM操作
--- 工具函数
--- 其他
